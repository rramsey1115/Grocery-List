# Grocery List App

This app was built using MERN stack, for purposes of demonstrating to the Persevere class at TTCC.

The purpose of building this app was to teach and model the following skills:
* creating client and server sides of application using the terminal
* installing all necessary dependencies using terminal
* organizing file structure
* creating new database in MongoDB Compas
* connecting backend code to MongoDB database instance
* communicating between client and server
* starting and stopping the app using terminal
* Creating React components
* Simple styling using Reactstrap and vanilla CSS
* Simple React hooks
* Simple CRUD implementation
  
### Instructor: Ryan Ramsey
